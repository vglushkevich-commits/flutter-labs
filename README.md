# Заметки - Простое приложение для ведения заметок

Простое мобильное приложение для создания и управления текстовыми заметками. Позволяет пользователям быстро записывать мысли, идеи и важную информацию.

## Основные функции

- Создание новых текстовых заметок
- Просмотр списка всех заметок
- Редактирование существующих заметок
- Удаление заметок
- Поиск по заметкам
- Получение случайных картинок котов (через внешнее API)

### 2. Экран создания/редактирования заметки

## Навигация
Главный экран
↓
[Список заметок] → [Создать новую] → Экран редактирования
↓
[Выбрать заметку] → Экран редактирования
↓
[Случайные котики] → Экран картинок с котами (API)
