# Заметки - Простое приложение для ведения заметок

Простое мобильное приложение для создания и управления текстовыми заметками. Позволяет пользователям быстро записывать мысли, идеи и важную информацию.

## Основные функции

- Создание новых текстовых заметок
- Просмотр списка всех заметок
- Редактирование существующих заметок
- Удаление заметок
- Поиск по заметкам
- Получение случайных цветовых палитр для оформления заметок (через внешнее API)
